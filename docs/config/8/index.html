限速规则

白山支持针对单个请求url进行限速，通过限速该功能，您可以对域名整体带宽峰值进行控制。

配置指引

  1. 登录控制台，在左侧菜单栏域名管理内选择【域名配置】，选择域名，再选择【添加配置】。点击【添加配置】，选择【特殊功能】内的添加【限速】

  2. 配置添加完成后点击【请点击发布】，再【确认保存】，配置即可生效

  3. 配置参数含义：

    • 前面多少字节不限速：为保障用户体验，限速规则可针对文件请求前一段不限速。例如用户请求100M的文件，可设置前12M不进行限速，12M之后的请求根据限速值限速。该参数为非必填项。如果不填写，默认值为0，表示整个文件都限速。
    • 限速值：限速值为必填项，参数值必须大于等于0。

  4. 注意事项：

    • 该功能是针对域名下的所有请求都进行限速，如果需要针对指定的url限速，请联系白山相关同事。

配置案例

域名Blog4.lowxp.com针对文件前5M不限速，超过5M后限速600KB/s

用户请求80M文件 http://Blog4.lowxp.com/2.mp4，则用户请求0-5M内不做限速，5M-80M限速600KB/s。
